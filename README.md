# Python All My Projects

Bu repo, Python ile yazılmış konsol uygulamalarından oluşan 7 adet eğitim projesi içermektedir. Projeler veritabanı işlemleri, UUID oluşturma, veri dönüştürme ve algoritma problemlerini kapsamaktadır.

## Proje Listesi

| # | Proje | Açıklama |
|---|-------|----------|
| 1 | Python_01_Mini_Projects | Mini Python projeleri (Excel to DB, JSON Creator, vb.) |
| 2 | Python_02_SQLite_Tutorial | SQLite3 veritabanı eğitimi (8 ders) |
| 3 | Python_03_SQLAlchemy_Tutorial | SQLAlchemy ORM eğitimi |
| 4 | Python_04_UUID_Creator | UUID oluşturucu |
| 5 | Python_05_UUID_Module_Kullanimi | UUID modülü kullanımı |
| 6 | Python_06_SQLite | SQLite veritabanı işlemleri |
| 7 | Python_07_20_Problem | 20 algoritma problemi çözümü |

## Kapsanan Konular

### Veritabanı İşlemleri
- SQLite3 ile veritabanı oluşturma ve yönetme
- SQLAlchemy ORM kullanımı
- CRUD işlemleri (Create, Read, Update, Delete)

### Veri İşleme
- Excel'den veritabanına veri aktarma
- JSON string oluşturma
- Insert query oluşturma

### Algoritmalar
- 20 farklı programlama problemi çözümü
- Temel algoritma pratikleri

### Yardımcı Araçlar
- UUID oluşturma ve kullanımı
- Metin işleme araçları

## Gereksinimler

- Python 3.x
- SQLite3 (built-in)
- SQLAlchemy (`pip install sqlalchemy`)

## Kullanım

Her proje klasörü bağımsız olarak çalıştırılabilir:

```bash
cd Python_01_Mini_Projects
python main.py
```
